# LIFT Method

NAUCode methodology IP asset. Measures project progress in **low-code/no-code**
contexts via Certainty Units, replacing the effort-based assumption that
breaks when AI generates near-infinite effort.

## Provenance

LIFT didn't start as a method. It started as NAUCode's practitioner-led
observation: Shape Up's Hill Chart didn't fit LCNC project reality. The deck
in `2ngnhan/may26ws/lift.html` traces the journey from "this isn't working" to
"here's the framework, here's the DSR audit."
